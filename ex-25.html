Webpage Figures and Captioning

Understanding how to use Figures

There are semantic elements that are typically used for images or even code blocks:

  • The <figure> elemment is used for graphics, images, or even a diagram.
  • Usually the contents in this element are grouped together and can be placed in the webpag's <main> element or elsewhere
  • For instance the code would look like this:
    • <figure> Figure content goes header </figure>
